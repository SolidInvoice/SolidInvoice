============
Introduction
============

SolidInvoice is an open-source application that allows you to manage clients and contacts and send invoices and quotes.

-------------------
System Requirements
-------------------

.. note::
    In order to run SolidInvoice, you need at least PHP version **7.3.0**, but the latest version is always recommended

**Requirements:**

* PHP (>=7.3.0)
* MySQL Database (> 5.6.0)
